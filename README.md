# Delivery-App-Doorsnap

Food Ordering App

Responsive web app for browsing a menu, adding dishes to a basket, and viewing totals with delivery costs. Optimized for mobile, tablet, and desktop, following a Figma design. Prices use comma notation.

Features

Category-based menu: Street Food, Bowls, Burgers

Add/remove items with +/- buttons

Dynamic basket updates (only affected buttons re-render)

Close basket via top-right X

Smooth scroll to sections

Responsive basket, footer, and mobile click areas
